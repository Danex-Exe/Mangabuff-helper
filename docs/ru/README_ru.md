# Mangabuff Quiz Auto Answer

Скрипт для Tampermonkey, который автоматически проходит квизы на [mangabuff.ru/quiz](https://mangabuff.ru/quiz), отправляя правильные ответы на каждый вопрос.

## Особенности

- Автоматически запускает квиз при загрузке страницы.
- Извлекает правильный ответ из ответа сервера.
- Отправляет ответ с настраиваемой задержкой.
- Проходит все вопросы до завершения квиза.

## Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера, если оно ещё не установлено.
2. Нажмите на иконку Tampermonkey и выберите **«Создать новый скрипт»**.
3. Замените код по умолчанию на полный код скрипта (см. ниже).
4. Сохраните скрипт (Ctrl+S или Файл → Сохранить).
5. Перейдите на страницу `https://mangabuff.ru/quiz` – скрипт запустится автоматически.

## Логирование

Скрипт выводит свои действия в консоль браузера (F12 → вкладка Console). Вы можете увидеть:
- Запуск квиза и первый ответ.
- Каждый отправленный ответ и ответ сервера.
- Задержки из‑за ограничения частоты запросов.
- Ошибки, если они возникают.

## Принцип работы

1. При загрузке страницы скрипт отправляет POST‑запрос на `/quiz/start`, чтобы начать квиз.
2. Из ответа извлекается правильный ответ (`correct_text`).
3. После небольшой начальной задержки (2 секунды) ответ отправляется на `/quiz/answer`.
4. Цикл повторяется, пока не будет получен ответ без следующего вопроса (квиз завершён).

## Отказ от ответственности

Используйте этот скрипт ответственно. Автоматизация действий на веб‑сайте может нарушать его правила использования. Скрипт предоставлен исключительно в образовательных целях.
